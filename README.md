## About The Project

This is the API that I developed utilizing the *Node.js* framework for the <a href="https://github.com/caglayagmuricerr/notes-app"><strong>Notable-Now</strong></a> application.

ðŸ“¨This API provides a bunch of special paths that the app can use to ask for specific things or do certain actions.

ðŸ”’Safety is also really important. Just like how you lock your front door, this API uses special codes to make sure only the right people can use it and play with the data. It uses hashed passwords (and hashed texts in the next version) as a security measure.

ðŸ“ˆAnd the cool part is that this API is built in a way that makes it easy to add more stuff later if the app needs to do new things.
